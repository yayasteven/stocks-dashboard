⚙ 雲端設定（GitHub Gist 同步）

設定後，所有資料會自動存到你的 GitHub Gist。任何裝置開這個網頁、輸入相同的 PAT 與 Gist ID 就能即時同步。未設定時走「離線模式」，使用內建 data.js。
